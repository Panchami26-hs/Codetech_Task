Project Overview

This project focuses on performing Big Data Analysis using PySpark in Jupyter Notebook. A large real-world dataset (Netflix Movies and TV Shows) was used to demonstrate data loading, processing, and extracting insights using big data tools.

 Tools & Technologies Used

Python

Jupyter Notebook

PySpark

Kaggle Dataset (Netflix Movies and TV Shows)

 Dataset what i Downloaded

Dataset Name: Netflix Movies and TV Shows
Format: CSV (netflix_titles.csv)
Source: Kaggle

 Steps Performed are

Installed required libraries (PySpark)

Created Spark session in Jupyter Notebook

Loaded the large dataset using Spark

Displayed and explored the dataset

Performed data analysis using:

GroupBy operations

Filtering operations

Extracted meaningful insights

Added conclusions

 Key Insights of the Task

The dataset contains both Movies and TV Shows.

The majority of content is produced in a few major countries.

Recent years show increased content production.
